# Equirectangular Storyboard Web

Equirectangular の絵コンテ編集ツール。

## 起動

```powershell
npm install
npm run dev
```

## EXE 作成

```powershell
npm run dist:win
```

`release/` に単体起動の portable exe が出ます。

## 主な機能

- 2D 画像の配置、回転、距離調整
- Equirectangular への投影プレビュー
- Three.js ベースの VR プレビュー
- シーンの保存、読込、最近のファイル一覧
