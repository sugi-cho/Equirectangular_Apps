# Equirectangular Storyboard Web

Equirectangular の絵コンテ編集ツール。

## 起動

```powershell
npm install
npm run dev
```

## ワンクリック起動

`start-webapp.bat` を実行すると、必要なら Node.js を入れてから依存を導入し、Web アプリを起動してブラウザで開きます。

## 主な機能

- 2D 画像の配置、回転、距離調整
- Equirectangular への投影プレビュー
- Three.js ベースの VR プレビュー
- シーンの保存、読込、最近のファイル一覧
